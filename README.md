MyModibleAssistant
==================
设置完成后,根据当前的程序,跳转到不同的程序<br/>
直接点击=跳转到第一个程序<br/>
快速左右划动=第二个<br/>
快速上下划动=第三个<br/>
双击=返回前一个<br/>
3击=退两步<br/>
